set_custom_top_level_module
-------------

This TCL command allows user to overwrite the name of top level entity in
Quartus compilation.

Category
+++++++++

Quartus

Value Type
+++++++++++

string

Dependencies
+++++++++++++

NONE

Applicable Flows
+++++++++++++++++

All devices and flows

Test Status
++++++++++++

Actively in-use

Examples
+++++++++

    ``set_custom_top_level_module foo``
