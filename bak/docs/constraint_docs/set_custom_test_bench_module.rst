.. _set_custom_test_bench_module:

set_custom_test_bench_module
-------------

This TCL command is to overwrite the name of testbench module to be elaborated
in simulation.

Category
+++++++++

Simulation

Value Type
+++++++++++

String

Dependencies
+++++++++++++

None

Applicable Flows
+++++++++++++++++

All devices and flows

Test Status
++++++++++++

Actively in-use

Examples
+++++++++

    ``set_custom_test_bench_module "custom_tb"``
